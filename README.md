# Curso de Git e GitHub - Curso em Vídeo

Este repositório armazena os exercícios, testes e projetos desenvolvidos durante o curso de Git e GitHub do professor Gustavo Guanabara (Curso em Vídeo). 

O objetivo aqui é praticar o versionamento de código, explorando desde os comandos básicos até fluxos de trabalho mais complexos.

## 🚀 O que estou praticando
- [ ] Configuração inicial do ambiente e Git Bash.
- [ ] Comandos essenciais: `git init`, `add`, `commit`, `status`.
- [ ] Trabalhando com repositórios remotos: `push`, `pull`, `clone`.
- [ ] Manipulação de ramos (branches) e resolução de conflitos.
- [ ] Testes de fluxo de trabalho e boas práticas de commit.

## 🛠️ Tecnologias Utilizadas
- Git
- GitHub
